数据分析中的业务指标

发表时间：2020-11-18 阅读： 评论：

什么是数据指标？

数据指标之指从原始数据中提炼出对当前业务具有参考价值的统计数据

数据指标分类

在互联网公司中，通常数据部门主要得到和研究以下三种数据：
用户数据，用户行为数据，产品数据
所以对应这三种数据产生三种数据指标如下：
用户数据指标，用户行为数据指标，产品数据指标

用户数据指标

用户数据：
通常包括以下字段
用户唯一ID
用户基本信息
时间戳
用户数据指标：

日新增用户

什么是日新增用户？
就是产品每天新增用户数量，更深层次理解其实是用户第一天访问软件的那天，这个用户就是当天新增用户
为什么需要关注新增用户那？
1.产品如果没有新增用户，或者新增用户少于减少用户，那么用户数就会慢慢减少
2.新增用户来自推广的渠道，如果按渠道划分，可以知道每个渠道新增了多少用户，从而判断出渠道推广效果，并指导后续推广方案
如何计算每天新增用户？
可以统计每个用户访问软件最早的日期，进行汇总，就可以得到日新增用户

活跃用户数

什么是活跃用户?
使用该软件，网站，或者服务的用户数量
活跃用户分类

日活跃用户数（DAU）

指每天zhi活跃用户数量

周活跃用户数（WAU)

指每周活跃用户数量

月活跃用户数（MAU）

指每月活跃用户数

活跃率

活跃率 = 活跃用户数/总用户数

留存率

什么是留存？
留存指经过推广获得的用户，在一段时间内还会继续使用该软件，网站，或者服务，这些用户就是留存用户
留存作用是什么？
可以反应产品用户黏性
什么是留存率？
留存率 = 第一天新增用户并且之后又出现的用户数/第一天新增用户总数

次日留存率

指当天新增用户中，第二天又出现的用户数/当天新增用户总数

第三日留存率

第一天新增用户中，在第三天又出现的用户数/当天新增用户总数

第七日留存率

第一天新增用户中，在第七天又出现的用户数/当天新增用户总数

第三十日留存率

第一天新增用户中，在第三十天又出现的用户数/当天新增用户总数

产品数据指标

产品数据：
文章标题
阅读量

产品数据指标：

常用专用名词汇总

DAU: Daily Active User 日活跃用户数
WAU: Weekly Active User 周活跃用户数
MAU: Monthly Active User 月活跃用户数
DNU: Day New User 日新增用户数
DOU: Day Old User 日老用户（非新增用户）
ACU: Average Concurrent User 平均同时在线人数
PCU: Peak Concurrent User 最高同时在线人数
UV: Unique Visitor 唯一访问量，既页面被多少人访问过
PV: Page View 页面浏览量
ARPU: Average Revenue Per User 平均每个活跃用户收益
ARPPU: Average Revenue Per Paying User 付费用户平均收益
LTV: Life Time Value 生命周期价值（产品从用户所有互动中获取的全部经济收益）
CAC: Customer Acquisition Cost 用户获取成本
ROI: Return On Investment 投资回报率 ROI = 利润总额/投入成本总额
GMV: Gross Merchandise Volume 成交总额 MGV = 销售额+取消订单金额+退款金额
支付UV:下单并成功支付的用户数
